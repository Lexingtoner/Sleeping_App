# Sleeping App

Консольное приложение на Kotlin, которое помогает расслабиться и заснуть с помощью встроенного "ИИ"-помощника.

## Запуск

1. Установите JDK 17.
2. Запустите приложение:

```bash
gradle run
```

Приложение задаст пару вопросов и предложит дыхательную практику, чтобы мягко подготовиться ко сну.
